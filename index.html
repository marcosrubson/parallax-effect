THE PARALLAX ON WEB

Parallax scrolling is a website trend where background content moves at a different speed than foreground content while scrolling. This effect is useful when we need to create the feeling of layers on the page and the difference in speed when viewing the images, thus creating a practical, yet sophisticated and visually pleasing effect for readers. This small example is based on lessons on the w3schools website.
